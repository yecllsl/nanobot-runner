# v0.21 数字孪生引擎 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 实现数字孪生引擎，构建5维度跑者状态向量，支持What-If推演和计划对比功能。

**Architecture:** 薄编排层方案。`DigitalTwinEngine` 聚合 `StateVectorBuilder` 和 `WhatIfSimulator`，复用 v0.20 的 `PredictionEngine`、`BodySignalEngine`、`BanisterIRModel` 等已有模块，不引入新状态转移引擎。

**Tech Stack:** Python 3.11+, dataclasses(frozen), Polars, numpy, scipy, Typer, Rich

### Task 6: CLI Handler — TwinHandler

**Files:**
- Create: `src/cli/handlers/twin_handler.py`

- [ ] **Step 1: 创建 src/cli/handlers/twin_handler.py**

遵循 `PredictionHandler` 模式：构造函数接受可选 `AppContext`，通过 `context.twin_engine` 获取引擎实例。

```python
from __future__ import annotations

from typing import Any

from src.core.base.context import AppContext, AppContextFactory


class TwinHandler:
    """数字孪生业务逻辑"""

    def __init__(self, context: AppContext | None = None) -> None:
        if context is None:
            context = AppContextFactory.create()
        self.context = context

    def _get_engine(self) -> Any:
        return self.context.twin_engine

    def get_runner_state(self) -> dict[str, Any]:
        engine = self._get_engine()
        result = engine.get_runner_state()
        return result.to_dict()

    def simulate_plan(self, plan_json: str, weeks: int | None = None) -> dict[str, Any]:
        import json

        from src.core.twin.models import HypotheticalPlan

        engine = self._get_engine()
        plan_data = json.loads(plan_json)
        plan = HypotheticalPlan(
            name=plan_data.get("name", "未命名计划"),
            weeks=[
                __import__("src.core.twin.models", fromlist=["WeeklyPlanSpec"]).WeeklyPlanSpec(**w)
                for w in plan_data.get("weeks", [])
            ],
            source=plan_data.get("source", "manual"),
            plan_id=plan_data.get("plan_id"),
        )
        result = engine.simulate_plan(plan, weeks=weeks)
        return result.to_dict()

    def simulate_plan_by_id(self, plan_id: str, weeks: int | None = None) -> dict[str, Any]:
        from src.core.twin.models import HypotheticalPlan, WeeklyPlanSpec

        engine = self._get_engine()
        plan = HypotheticalPlan(
            name="", weeks=[], source="plan_id", plan_id=plan_id
        )
        result = engine.simulate_plan(plan, weeks=weeks)
        return result.to_dict()

    def compare_plans(self, plan_jsons: list[str]) -> dict[str, Any]:
        import json

        from src.core.twin.models import HypotheticalPlan, WeeklyPlanSpec

        engine = self._get_engine()
        plans: list[HypotheticalPlan] = []
        for pj in plan_jsons:
            plan_data = json.loads(pj)
            plans.append(
                HypotheticalPlan(
                    name=plan_data.get("name", "未命名计划"),
                    weeks=[
                        WeeklyPlanSpec(**w) for w in plan_data.get("weeks", [])
                    ],
                    source=plan_data.get("source", "manual"),
                    plan_id=plan_data.get("plan_id"),
                )
            )
        result = engine.compare_plans(plans)
        return result.to_dict()
```
