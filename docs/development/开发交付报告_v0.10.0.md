## 5. 启动方式

```bash
# 记录执行反馈
uv run nanobotrun plan log <plan_id> <date> --completion 0.8 --effort 6

# 查看执行统计
uv run nanobotrun plan stats <plan_id>

# Agent交互
uv run nanobotrun agent chat
```
