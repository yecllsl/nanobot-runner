## 5. 架构说明

### 5.1 技能管理架构

```
config.json (skills.disabled配置)
       ↓
SkillManager.list_skills()
       ↓
扫描skills/目录，解析SKILL.md
       ↓
返回SkillInfo列表
       ↓
CLI命令展示/启用/禁用技能
```

### 5.2 技能文件格式

```yaml
---
name: skill-name
description: 技能描述
version: 1.0.0
author: author-name
tags:
  - tag1
  - tag2
dependencies:
  - name: dep-name
    version: "1.0"
    optional: true
enabled_tools:
  - mcp_server_tool_name
---

# 技能内容

技能的详细说明和使用指南...
```

### 5.3 CLI命令设计

| 命令 | 功能 |
|------|------|
| `nanobotrun skill list` | 列出所有可用技能 |
| `nanobotrun skill enable <name>` | 启用指定技能 |
| `nanobotrun skill disable <name>` | 禁用指定技能 |
| `nanobotrun skill import <path>` | 导入自定义技能 |
| `nanobotrun skill show <name>` | 显示技能详细信息 |

## 6. 已知问题与注意事项

1. **技能目录**: 技能需要放置在`workspace/skills/`目录下
2. **SKILL.md格式**: 技能文件必须是`SKILL.md`，支持YAML front matter
3. **依赖管理**: 技能依赖目前仅用于展示，未实现自动安装
