## 六、注意事项

### 6.1 使用注意

1. **workspace 目录**: 默认为 `~/.nanobot-runner`，可通过构造函数参数自定义
2. **目录自动创建**: 初始化时会自动创建 `data/` 和 `memory/` 目录
3. **Agent 笔记标记**: 使用 `- @agent ` 格式标记 Agent 观察笔记
4. **同步方向**: 
   - `json_to_md`: 仅保存 MD（不保存 JSON）
   - `md_to_json`: 保存 JSON 并提取 Agent 笔记
   - `bidirectional`: 双向同步

### 6.2 兼容性

- Python >= 3.11（nanobot-ai 要求）
- Windows/Mac/Linux 跨平台兼容
- 与现有 ProfileEngine 类完全兼容

### 6.3 已知问题

- Windows 上难以测试"无效路径初始化"场景（需要管理员权限），该测试已跳过
