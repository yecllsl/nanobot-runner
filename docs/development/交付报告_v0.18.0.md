## 6. 关键设计决策

1. **降级策略**: 所有图表渲染均包含降级逻辑（纯文字表格），确保终端兼容性
2. **依赖注入**: AppContext 懒加载 `chart_renderer` 和 `export_engine`，与现有模式一致
3. **路径安全**: ExportEngine 拒绝 `../` 路径穿越，文件权限限制为仅当前用户可读写
4. **分批计算**: TSS/VDOT 计算采用 BATCH_SIZE=100，避免大数据量内存压力
5. **编码兼容**: CSV 使用 UTF-8 BOM 编码，确保 Excel 正常打开无乱码
