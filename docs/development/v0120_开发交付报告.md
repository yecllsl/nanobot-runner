## 5. 核心组件说明

### 5.1 GoalPredictionEngine

基于VDOT趋势、训练负荷和执行反馈，预测目标达成概率。

- 支持目标类型：vdot / 5k / 10k / half_marathon / marathon
- 核心方法：`evaluate_goal()` → 返回 `GoalAchievementEvaluation`
- 关键特性：VDOT趋势斜率计算、周提升率约束、置信度评估、风险识别

### 5.2 LongTermPlanGenerator

根据用户体能水平和目标，生成多周期训练规划。

- 周期类型：base(基础期) → build(提升期) → peak(巅峰期) → taper(减量期)
- 核心方法：`generate_plan()` → 返回 `LongTermPlan`
- 关键特性：周期分配比例、跑量范围计算、里程碑生成

### 5.3 SmartAdviceEngine

生成训练/恢复/营养/伤病预防四维度个性化建议。

- 建议类型：training / recovery / nutrition / injury_prevention
- 核心方法：`generate_advice()` → 返回 `list[SmartTrainingAdvice]`
- 关键特性：按优先级排序、多维度覆盖、上下文感知

## 6. CLI命令使用方式

```bash
# 目标达成评估
uv run nanobotrun plan evaluate vdot 50.0 --vdot 42.0 --weeks 16

# 创建长期训练规划
uv run nanobotrun plan long-term "半马备赛" --vdot 42.0 --target 48.0 --race "北京马拉松" --date 2026-10-15

# 获取智能训练建议
uv run nanobotrun plan advice --vdot 42.0 --volume 35 --risk low
```

## 8. 已知问题与限制

1. **VDOT趋势预测**：当历史数据点少于2个时，默认使用保守的周提升率(0.3)
2. **目标时间转换**：5k/10k/half_marathon/marathon目标值需以秒为单位传入
3. **建议引擎**：当前建议基于规则生成，后续可接入LLM增强个性化
