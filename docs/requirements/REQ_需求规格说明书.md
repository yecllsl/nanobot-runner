# 需求规格说明书

## 1. 项目概述

### 1.1 项目背景

Nanobot Runner 是一款桌面端私人 AI 跑步助理，基于 nanobot-ai 框架构建，旨在帮助跑步爱好者管理和分析运动数据。

### 1.2 项目目标

- 提供本地化的跑步数据管理方案
- 实现智能化的运动数据分析
- 构建自然语言交互的 AI 助理
- 保护用户数据隐私

### 1.3 目标用户

- 跑步爱好者
- 马拉松训练者
- 运动数据分析师

## 2. 功能需求

### 2.1 数据导入功能

#### 2.1.1 FIT文件解析

**需求描述**: 支持解析标准 FIT 格式的运动数据文件

**功能要点**:
- 解析 Garmin 等设备导出的 FIT 文件
- 提取运动数据（距离、时长、心率、配速等）
- 支持批量导入

**验收标准**:
- 成功解析标准 FIT 文件
- 数据提取完整准确
- 导入速度 < 1秒/文件

#### 2.1.2 数据去重

**需求描述**: 避免重复导入相同运动记录

**功能要点**:
- 基于 SHA256 指纹去重
- 支持强制重新导入
- 记录导入历史

**验收标准**:
- 去重准确率 100%
- 指纹计算速度 < 100ms/文件

### 2.2 数据存储功能

#### 2.2.1 Parquet存储

**需求描述**: 使用 Parquet 格式存储运动数据

**功能要点**:
- 按年份分片存储
- Snappy 压缩
- Schema 规范化

**验收标准**:
- 存储格式符合 Parquet 规范
- 压缩率 > 50%
- 查询性能 < 100ms

#### 2.2.2 数据索引

**需求描述**: 维护数据索引以支持快速查询

**功能要点**:
- 指纹索引
- 时间索引
- 距离索引

**验收标准**:
- 索引查询 < 10ms
- 索引更新 < 50ms

### 2.3 数据分析功能

#### 2.3.1 VDOT计算

**需求描述**: 计算跑步能力值（VDOT）

**功能要点**:
- 基于 Powers 公式计算
- 距离要求 >= 1500m
- 提供趋势分析

**验收标准**:
- 计算结果误差 < 1%
- 支持历史趋势查询

#### 2.3.2 训练负荷分析

**需求描述**: 计算训练压力和体能状态

**功能要点**:
- TSS（训练压力分数）计算
- ATL（急性训练负荷）计算
- CTL（慢性训练负荷）计算
- TSB（训练压力平衡）计算

**验收标准**:
- 计算结果符合运动科学标准
- 支持自定义时间窗口

#### 2.3.3 心率漂移分析

**需求描述**: 分析运动过程中心率变化趋势

**功能要点**:
- 计算心率与时间的相关性
- 判断心率漂移程度
- 提供有氧能力评估

**验收标准**:
- 相关性计算准确
- 漂移判断阈值可配置

### 2.4 Agent交互功能

#### 2.4.1 自然语言查询

**需求描述**: 支持自然语言查询运动数据

**功能要点**:
- 意图识别
- 参数提取
- 结果格式化

**验收标准**:
- 意图识别准确率 > 90%
- 响应时间 < 3秒

#### 2.4.2 智能建议

**需求描述**: 基于数据提供训练建议

**功能要点**:
- 训练强度建议
- 恢复时间建议
- 目标达成建议

**验收标准**:
- 建议符合运动科学原理
- 建议可执行性强

### 2.5 CLI功能

#### 2.5.1 命令行界面

**需求描述**: 提供完整的命令行操作界面

**功能要点**:
- 数据导入命令
- 统计查询命令
- Agent交互命令

**验收标准**:
- 命令响应 < 2秒
- 输出格式清晰易读

## 4. 约束条件

### 4.1 技术约束

- Python 3.11+
- 本地部署，无云服务依赖
- 支持 Windows/Linux/macOS

### 4.2 业务约束

- 仅支持 FIT 格式文件
- 单用户使用场景
- 数据存储在本地

### 4.3 法规约束

- 遵守数据保护法规
- 用户数据所有权归用户
- 不收集用户隐私数据

## 6. 数据需求

### 6.1 数据字典

| 数据项 | 类型 | 说明 |
|--------|------|------|
| activity_id | string | 活动唯一标识 |
| timestamp | datetime | 活动时间戳 |
| total_distance | float | 总距离（米） |
| total_timer_time | float | 总时长（秒） |
| avg_speed | float | 平均速度 |
| avg_heart_rate | float | 平均心率 |
| total_calories | float | 消耗卡路里 |
| vdot | float | VDOT值 |

### 6.2 数据量估算

| 数据类型 | 年增长量 |
|---------|---------|
| 运动记录 | ~500条/年 |
| 存储空间 | ~50MB/年 |
