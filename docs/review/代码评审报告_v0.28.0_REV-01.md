## 6. 问题清单

```
[P1] Token签发端点安全增强建议
File: src/core/webui/app.py:76
Issue: /api/auth/token 端点无任何认证保护，任何能访问该服务的人均可签发token。
       虽然当前默认绑定127.0.0.1（仅本地访问），但风险在于配置变更后可被外部访问。
Fix: 建议添加请求频率限制（rate limiting），或使用 Starlette 的 BaseHTTPMiddleware 
     添加 IP 白名单检查，限制仅本地访问可签发 token。
```

```
[P2] 模块路径与架构设计不一致
File: src/core/webui/ (vs 架构设计的 src/webui/)
Issue: 架构设计说明书中定义为 src/webui/，实际实现为 src/core/webui/。 
       路径差异不影响功能，但可能使开发者查找代码时产生混淆。
Fix: 选择其一：更新架构文档匹配实际路径，或将代码移至 src/webui/。
```

```
[P2] 路由文件名与架构设计不一致
File: src/core/webui/routes/body_signal.py (vs 架构设计的 body_signals.py)
Issue: 架构设计中命名为 body_signals.py（复数），实际命名为 body_signal.py（单数）。
Fix: 统一命名。
```
