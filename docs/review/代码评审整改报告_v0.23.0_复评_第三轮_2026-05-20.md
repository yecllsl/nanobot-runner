## 2. 整改明细

### M-01: `_extract_plan_id` 键名不一致

| 项目 | 内容 |
|------|------|
| **问题** | `_extract_plan_id` 使用 `call.get("args", {})` 提取参数，但 `DecisionLogHook` 写入的键名为 `"arguments"` |
| **影响** | `check_plan_execution()` 无法从 tool_call_chain 提取 plan_id，导致忠实度计算失败 |
| **修复** | 将 `call.get("args", {})` 改为 `call.get("arguments", {})` |
| **文件** | [outcome_collector.py:L480](file:///d:/yecll/Documents/LocalCode/RunFlowAgent.worktrees/feature-0.23.0/src/core/evolution/outcome_collector.py#L480) |

**修复前**:
```python
# 从tool_call_chain的args中提取
for call in decision.tool_call_chain:
    args = call.get("args", {})
```

**修复后**:
```python
# 从tool_call_chain的arguments中提取
for call in decision.tool_call_chain:
    args = call.get("arguments", {})
```

**测试数据同步修复**: 所有测试文件中 `tool_call_chain` 的 `"args"` 键统一改为 `"arguments"`，涉及：
- test_outcome_collector.py (4处)
- test_evolution_engine.py (1处)
- test_evolution_store.py (3处)
- test_decision_logger.py (1处)
- test_models.py (2处)

### M-02: Handler 突破封装访问私有成员

| 项目 | 内容 |
|------|------|
| **问题** | `evolution_handler.py` 直接调用 `engine.outcome_collector._compute_accuracy_stats()` 和访问 `engine.outcome_collector._store.query_outcomes()` |
| **影响** | 违反封装原则，Handler 与 OutcomeCollector 内部实现耦合 |
| **修复** | 在 OutcomeCollector 新增 `get_accuracy_stats()` 和 `get_fidelity_stats()` 公开方法，Handler 通过公开方法调用 |
| **文件** | [outcome_collector.py:L469-L517](file:///d:/yecll/Documents/LocalCode/RunFlowAgent.worktrees/feature-0.23.0/src/core/evolution/outcome_collector.py#L469-L517), [evolution_handler.py:L121-L137](file:///d:/yecll/Documents/LocalCode/RunFlowAgent.worktrees/feature-0.23.0/src/cli/handlers/evolution_handler.py#L121-L137) |

**新增公开方法**:

1. `OutcomeCollector.get_accuracy_stats(days: int = 30) -> PredictionAccuracyStats`
   - 委托 `_compute_accuracy_stats()` 实现
   - `days` 参数预留，当前为全量统计

2. `OutcomeCollector.get_fidelity_stats(days: int = 30) -> dict[str, float | int]`
   - 从存储层获取结果记录，按日期过滤
   - 返回 count/avg_fidelity/min_fidelity/max_fidelity

**Handler 修复前**:
```python
# get_accuracy
stats = engine.outcome_collector._compute_accuracy_stats()

# get_fidelity
outcomes = engine.outcome_collector._store.query_outcomes(limit=10000)
# ... 25行内联计算逻辑
```

**Handler 修复后**:
```python
# get_accuracy
stats = engine.outcome_collector.get_accuracy_stats(days=days)

# get_fidelity
return engine.outcome_collector.get_fidelity_stats(days=days)
```
