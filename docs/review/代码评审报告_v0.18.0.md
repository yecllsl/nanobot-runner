[P1] ExportEngine 中捕获裸 Exception 违反项目规范

文件: [src/core/export/engine.py](file:///d:/yecll/Documents/LocalCode/RunFlowAgent/src/core/export/engine.py#L175-L183)

问题: `export_sessions()` 和 `export_summary()` 方法中使用 `except Exception as e` 捕获所有异常。项目规范明确禁止使用裸 Exception，应使用项目自定义异常或更具体的异常类型。

当前代码:
```python
except Exception as e:
    duration_ms = int((time.perf_counter() - start_time) * 1000)
    return ExportResult(
        success=False,
        record_count=0,
        file_path=None,
        message=f"导出失败：{e}",
        duration_ms=duration_ms,
    )
```

修复建议: 将 `except Exception` 替换为 `except (StorageError, ValueError, OSError) as e`，仅捕获已知可能发生的异常类型。对于确实需要兜底的场景，添加 `# nosec` 注释说明原因。

[P1] VizHandler._render_text_table 硬编码终端宽度

文件: [src/cli/handlers/viz_handler.py](file:///d:/yecll/Documents/LocalCode/RunFlowAgent/src/cli/handlers/viz_handler.py#L167)

问题: `Console(force_terminal=True, width=100)` 硬编码宽度为 100，未使用 `_get_terminal_width()` 自适应终端宽度。当终端宽度小于 100 列时，表格可能显示异常。

当前代码:
```python
console = Console(force_terminal=True, width=100)
```

修复建议: 复用 `plotext_renderer._get_terminal_width()` 或 `shutil.get_terminal_size()` 动态获取终端宽度：
```python
from shutil import get_terminal_size
try:
    width = get_terminal_size().columns
except OSError:
    width = 80
console = Console(force_terminal=True, width=max(60, min(120, width)))
```

[P2] JsonExporter.validate_output_path 中无效的权限检查

文件: [src/core/export/json_exporter.py](file:///d:/yecll/Documents/LocalCode/RunFlowAgent/src/core/export/json_exporter.py#L113-L123)

问题: `if parent.exists() and not parent.stat().st_mode:` — `st_mode` 在 Windows 上始终非零，在 Unix 上也几乎不会为 0。此检查无法有效判断写权限，属于无效代码。

当前代码:
```python
if parent.exists() and not parent.stat().st_mode:
    return False
```

修复建议: 替换为实际的写权限检查：
```python
if parent.exists() and not os.access(str(parent), os.W_OK):
    return False
```
