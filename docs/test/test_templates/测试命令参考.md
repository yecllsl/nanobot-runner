# 测试命令参考

## 全部测试
```bash
uv run pytest tests/ -v
```

## 按类型执行
```bash
uv run pytest tests/unit/ -v           # 单元测试
uv run pytest tests/integration/ -v    # 集成测试
uv run pytest tests/e2e/ -v            # E2E测试
uv run pytest tests/performance/ -v    # 性能测试
```

## 按模块执行
```bash
uv run pytest tests/unit/core/ -v      # 核心模块测试
uv run pytest tests/integration/scene/ -v  # 场景集成测试
uv run pytest tests/unit/planning/ -v  # 智能跑步计划单元测试
```

## 覆盖率报告
```bash
uv run pytest tests/ --cov=src --cov-report=term-missing
```

## 特定测试
```bash
uv run pytest -k "test_import" -v      # 关键字匹配
uv run pytest tests/unit/test_parser.py -v  # 指定文件
```

## 代码质量
```bash
uv run ruff check src/ tests/
uv run ruff format --check src/ tests/
uv run mypy src/ --ignore-missing-imports
```

## 本地预检查（Windows PowerShell）
```powershell
uv run ruff format --check src/ tests/; if($?) {
    uv run ruff check src/ tests/; if($?) {
        uv run mypy src/ --ignore-missing-imports; if($?) {
            uv run pytest tests/unit/ --cov=src
        }
    }
}
```

## 准入验证命令
```bash
uv run ruff check src/ tests/
uv run ruff format --check src/ tests/
uv run mypy src/ --ignore-missing-imports
uv run pytest tests/unit/ --cov=src --cov-report=term-missing
```

## 准出验证命令
```bash
uv run pytest tests/ -v --tb=short
uv run pytest tests/integration/ -v
uv run pytest tests/e2e/ -v
uv run pytest tests/performance/ -v
```
