# UAT Gateway 与 Agent Chat 增强测试用例（UAT-101 ~ UAT-115）

> **版本**: v1.0（v0.22.0 质量收口补充版）
> **创建日期**: 2026-05-17

## 测试范围说明

本文件补充以下测试维度：
- **Gateway 异常场景**（UAT-101 ~ UAT-106）：LLM API 不可用、通道连接失败、MCP 工具不可用、超长消息等
- **Agent Chat 增强**（UAT-107 ~ UAT-112）：无LLM配置、上下文保持、无效输入、工具调用失败等
- **性能与边界**（UAT-113 ~ UAT-115）：大数据量查询、长时间运行、消息队列积压
