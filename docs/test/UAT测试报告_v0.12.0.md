## 四、缺陷统计

### 4.1 缺陷列表

| 缺陷ID | 严重等级 | 所属模块 | 缺陷描述 | 状态 |
|--------|----------|----------|----------|------|
| BUG-001 | 一般 | 训练计划 | CLI缺少创建TrainingPlan的命令，导致UAT-013/014无法执行 | 待修复 |

### 4.2 缺陷详情

**BUG-001: CLI缺少创建TrainingPlan的命令**
- **模块**: 训练计划
- **严重等级**: 一般（非核心功能异常，不影响主流程）
- **复现步骤**: 
  1. 执行`uv run nanobotrun plan long-term`创建长期规划
  2. 执行`uv run nanobotrun plan log <plan_id>`记录反馈
  3. 报错：计划不存在
- **根因分析**: `plan long-term`创建的是`LongTermPlan`（长期规划），而`plan log`需要的是`TrainingPlan`的训练计划ID。两者是不同的数据模型，当前CLI没有提供创建`TrainingPlan`的命令。
- **修复建议**: 
  - 方案A：新增`plan create`命令，用于创建TrainingPlan
  - 方案B：修改`plan log`命令，支持直接使用日期记录反馈，无需关联计划
  - 方案C：在`plan long-term`执行后自动创建关联的TrainingPlan
