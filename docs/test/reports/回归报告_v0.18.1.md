## 5. 已知问题与注意事项

1. **代码规范**: `plotext_renderer.py` 存在 1 个 ruff SIM105 优化建议，建议使用 `contextlib.suppress(OSError, ValueError)` 替代 `try-except-pass`，不影响功能
2. **编码范围**: `_ensure_utf8_output()` 仅在图表渲染时生效，其他 CLI 输出的中文编码问题可能仍需在更上层统一处理
3. **可视化导出**: 当前导出为纯文本格式，后续可考虑支持 SVG/PNG 等图形格式
4. **VDOT 聚合**: 同日多记录取平均值，未来可考虑提供聚合策略选项
