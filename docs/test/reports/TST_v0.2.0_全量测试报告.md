## 四、Bug清单

### 4.1 致命级Bug (P0)

#### Bug ID: BUG-001
**标题**: StorageManager缺少save_activities方法
**所属模块**: `src/core/storage.py`
**严重等级**: 致命
**优先级**: P0
**出现版本**: v0.2.0

**复现步骤**:
1. 执行集成测试中的存储相关测试
2. 调用StorageManager的save_activities方法
3. 出现AttributeError异常

**实际结果**:
```
AttributeError: 'StorageManager' object has no attribute 'save_activities'
```

**预期结果**:
- StorageManager应提供save_activities方法用于保存活动数据

**根因分析**:
- 方法命名不一致，实际方法名为`save_data`
- 集成测试用例使用了错误的方法名

**修复建议**:
1. 统一方法命名规范
2. 更新集成测试用例
3. 添加方法存在性验证

#### Bug ID: BUG-002
**标题**: CLI命令中文输出编码问题
**所属模块**: `src/cli.py`
**严重等级**: 致命
**优先级**: P0
**出现版本**: v0.2.0

**复现步骤**:
1. 执行`uv run nanobotrun stats`命令
2. 观察输出内容
3. 出现编码错误或乱码

**实际结果**:
```
UnicodeDecodeError: 'utf-8' codec can't decode byte 0xbb in position 895: invalid start byte
```

**预期结果**:
- CLI命令应正确显示中文内容
- 无编码错误

**根因分析**:
- Windows环境下编码处理不一致
- 子进程通信编码设置错误

**修复建议**:
1. 统一使用UTF-8编码
2. 设置正确的环境变量
3. 添加编码异常处理
