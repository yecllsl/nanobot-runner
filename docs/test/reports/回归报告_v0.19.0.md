## 6. 已知问题与注意事项

1. **HRV估算精度**: 当前HRV估算基于心率数据推导，无RR间期时标注"非医疗级精度"，属设计预期
2. **疲劳度评分**: 无RPE输入时权重重分配可能影响评分敏感度，属设计预期
3. **休息日效果判定**: 当前基于静息心率和TSB变化，未来可结合睡眠质量等多维度数据
4. **覆盖率优化**: `hrv_analyzer.py`(77%) 和 `body_signal_engine.py`(79%) 覆盖率略低于85%目标，但整体模块86%已达标，建议后续补充边界用例
