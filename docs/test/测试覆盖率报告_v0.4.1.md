## CI质量门禁配置

### 覆盖率检查配置

```yaml
# .github/workflows/ci.yml 覆盖率检查配置
coverage:
  status:
    project:
      default:
        target: 80%        # 总体目标
        threshold: 2%      # 允许下降阈值
      core:
        target: 80%        # 核心模块目标
        paths:
          - "src/core/"
      agents:
        target: 70%        # Agent模块目标
        paths:
          - "src/agents/"
      cli:
        target: 60%        # CLI模块目标
        paths:
          - "src/cli*.py"
    patch:
      default:
        target: 80%        # 新增代码目标
```

### 质量门禁规则

| 检查项 | 阈值 | 失败处理 |
|--------|------|----------|
| 总体覆盖率 | ≥80% | 阻止合并 |
| 核心模块覆盖率 | ≥80% | 阻止合并 |
| 新增代码覆盖率 | ≥80% | 阻止合并 |
| CLI模块覆盖率 | ≥60% | 警告（v0.4.2起阻止） |
