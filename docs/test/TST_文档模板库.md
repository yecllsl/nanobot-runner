## 一、模板使用说明

### 1.1 模板分类

本模板库包含以下类型的测试文档模板：

| 模板类型 | 用途 | 使用时机 | 负责人 |
|---------|------|---------|--------|
| 测试计划模板 | 规划测试活动 | 项目启动阶段 | 测试经理 |
| 测试用例模板 | 设计具体测试 | 测试设计阶段 | 测试工程师 |
| 测试报告模板 | 报告测试结果 | 测试执行后 | 测试工程师 |
| 缺陷报告模板 | 记录和跟踪缺陷 | 缺陷发现时 | 测试工程师 |
| 测试验收模板 | 确认测试完成 | 上线前 | 测试经理 |

### 1.2 模板使用规范

1. **复制模板**: 将模板内容复制到新文件中
2. **修改占位符**: 替换 `[占位符]` 为实际内容
3. **保持格式**: 保持原有的格式和结构
4. **版本控制**: 文档需要纳入版本控制

## 七、模板使用指南

### 7.1 模板选择指南

根据测试阶段选择合适的模板：

| 测试阶段 | 主要模板 | 辅助模板 |
|---------|---------|---------|
| 测试规划 | 测试计划模板 | - |
| 测试设计 | 测试用例模板 | - |
| 测试执行 | 缺陷报告模板 | 测试用例模板 |
| 测试报告 | 测试报告模板 | 缺陷报告模板 |
| 测试验收 | 测试验收模板 | 测试报告模板 |

### 7.2 模板填写注意事项

1. **及时性**: 测试文档需要及时更新和填写
2. **准确性**: 所有数据需要准确无误
3. **完整性**: 确保模板所有部分都填写完整
4. **一致性**: 保持文档之间的数据一致性
5. **可追溯性**: 确保所有操作都有记录

### 7.3 模板版本管理

- 每次重大修改需要更新版本号
- 保留历史版本用于追溯
- 模板变更需要经过评审
