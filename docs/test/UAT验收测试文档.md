## 1. 测试目的

本UAT文档旨在验证 Nanobot Runner v0.2.0 版本是否满足用户需求规格说明书中的功能要求，确保系统在三端（CLI、Chat、飞书）环境下稳定运行，为用户提供可靠的跑步数据分析服务。

### 1.1 验收目标

1. 验证三端核心功能是否符合需求规格说明书
2. 验证系统稳定性和可靠性
3. 验证用户体验是否达到预期
4. 验证数据准确性是否符合标准
5. 确认系统无致命/严重缺陷

### 1.2 验收标准

| 等级    | 标准                       |
| ----- | ------------------------ |
| 通过    | 所有P0级测试用例100%通过，无致命/严重缺陷 |
| 有条件通过 | P0级测试用例≥95%通过，严重缺陷≤1个    |
| 不通过   | P0级测试用例<95%通过，或存在致命缺陷    |

## 3. 测试环境配置

### 3.1 硬件环境

| 配置项 | 要求                  |
| --- | ------------------- |
| CPU | Intel Core i5 或同等性能 |
| 内存  | ≥8GB                |
| 硬盘  | ≥10GB可用空间           |

### 3.2 软件环境

| 软件     | 版本要求                      |
| ------ | ------------------------- |
| 操作系统   | Windows 10/11 或 macOS 12+ |
| Python | 3.11+                     |
| uv     | 最新版本                      |

### 3.3 测试数据

| 数据类型  | 数量   | 说明             |
| ----- | ---- | -------------- |
| FIT文件 | 10个  | 包含不同距离、时长的跑步记录 |

```bash

# ===== 环境准备 =====
uv sync                                    # 同步依赖

uv run nanobotrun import test.fit          # 导入FIT文件

uv run nanobotrun stats --year 2024        # 年度统计
uv run nanobotrun stats --start 2024-01-01 --end 2024-12-31  # 时间段统计
uv run nanobotrun vdot                     # VDOT趋势
uv run nanobotrun vdot --limit 10          # 最近10次VDOT
uv run nanobotrun load                     # 训练负荷
uv run nanobotrun load --days 30           # 30天负荷趋势
uv run nanobotrun recent                   # 最近10次训练记录
uv run nanobotrun recent -n 5              # 最近5次训练记录
uv run nanobotrun hr-drift                 # 心率漂移
uv run nanobotrun report                   # 生成晨报
uv run nanobotrun report --push            # 推送到飞书
uv run nanobotrun report --schedule 07:00  # 定时推送
uv run nanobotrun profile show             # 用户画像
uv run nanobotrun gateway                  # 启动飞书Gateway
uv run nanobotrun version                  # 显示版本号
uv run nanobotrun memory show              # 显示Agent记忆
uv run nanobotrun memory clear             # 清空Agent记忆
uv run nanobotrun init                     # 初始化工作区

# ===== Chat端测试 =====
uv run nanobotrun chat                     # 启动Chat模式
# 然后输入自然语言查询

# ===== 测试执行 =====
uv run pytest                              # 全量测试
uv run pytest tests/unit/                  # 单元测试
uv run pytest tests/integration/           # 集成测试
uv run pytest tests/e2e/                   # E2E测试
uv run pytest --cov=src --cov-report=html  # 覆盖率报告
```
