# Agent Chat 模式用户验收测试文档

**文档版本**: v1.0\
**创建日期**: 2026-04-13\
**适用版本**: v0.9.0+\
**测试类型**: 用户验收测试（UAT）\
**测试模式**: Agent Chat 自然语言交互模式

***

## 一、测试说明

### 1.1 测试目的

验证 Agent Chat 模式是否满足用户需求，确保：

- 自然语言交互流畅、准确
- 数据查询功能完整
- 训练建议专业、合理
- 用户体验良好

### 1.2 测试环境要求

| 项目     | 要求                           |
| ------ | ---------------------------- |
| 操作系统   | Windows 10/11 或 macOS 10.15+ |
| Python | 3.11 或更高版本                   |
| 网络环境   | 需要网络连接（用于 AI 模型调用）           |
| 测试数据   | 至少准备 7-14 天的跑步记录             |
| AI 配置  | 已配置本地模型或 API Key             |

### 1.3 测试前准备

#### 1.3.1 环境准备

```bash
# 1. 进入项目目录
cd nanobot-runner

# 2. 激活虚拟环境
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # macOS/Linux

# 3. 确认依赖已安装
uv sync --all-extras
```

#### 1.3.2 数据准备

```bash
# 1. 导入测试数据（至少 7-14 天的跑步记录）
uv run nanobotrun data import <FIT文件路径>

# 2. 验证数据已导入
uv run nanobotrun data stats
```

#### 1.3.3 AI 配置检查

```bash
# 检查 AI 配置是否正确
uv run nanobotrun system config
```

**预期输出**:

- 显示 AI 模型配置信息
- 显示 API Key 状态（如使用云端模型）

- "我今天应该跑吗？"

### 4.2 Agent Chat 常用命令

```bash
# 启动 Agent Chat
```
