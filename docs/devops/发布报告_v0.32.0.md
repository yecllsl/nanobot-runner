## 版本主题

**nanobot-ai 0.2.2 升级与配置分离** —— 移除 monkey-patch 技术债，实现配置分离，新增 SDK 编程式调用、运行时事件总线、自定义 Provider、语音转录配置

## 核心变更

### 底座升级
- nanobot-ai 0.2.1 → 0.2.2
- 支持 ProviderConfig 对象（apiBase 字段）
- 运行时事件总线集成

### 配置分离
- config.json → nanobot_config.json
- nanobot_config.json 作为 nanobot 配置唯一来源
- 移除运行时转换层，消除配置重复定义

### 技术债清偿
- 移除 4 项 monkey-patch
- ConfigInjector 替代运行时配置注入

### 新增组件
- **ConfigInjector**: 配置注入器，替代 monkey-patch
- **SDKAdapter**: SDK 编程式调用接口
- **RuntimeEventHook**: 运行时事件订阅 Hook
- **DynamicProviderRegistry**: 自定义 Provider 注册
- **AgentLoopAdapter**: AgentLoop 封装层

### 新功能
- SDK 编程式调用（同步/异步）
- 运行时事件 SSE 端点
- 自定义 Provider 支持
- 语音转录配置
