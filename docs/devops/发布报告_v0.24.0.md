## 2. 变更内容

### 新增功能

#### 训练响应性分析（REQ-0.24-01, P0）
- 分析不同训练类型（Easy/Tempo/Interval/LongRun）对 VDOT 提升的效果排名
- 基于 fidelity >= 0.7 的决策-结果配对数据
- 输出个人训练响应画像（profile_summary）
- 三级优先级推断（结构化数据 > 关键词匹配 > 兜底）
- 最低样本数门槛（每组 >= 5）

#### 预测校准层（REQ-0.24-02, P1）
- 偏差检测（mean_absolute_error / mean_bias）
- EMA 自适应更新（alpha = 0.3）
- 幅度限制（-20% ~ +20%）
- 重置功能（reset_to_default）
- 样本门槛 >= 10 条配对

#### 个性化模型进化（REQ-0.24-03, P1）
- Banister IR 参数调整（d1/d2/g1/g2/taud1/taud2/taug1/taug2）
- 风险阈值调整（injury_risk_threshold）
- JSON 持久化（calibrations/ 目录）
- 双向调整（增加/减少）

### 技术改进
- **Agent 工具新增**: `analyze_training_response`, `get_calibration_status`
- **CLI 命令新增**: `evolution response`, `evolution calibration`
- **依赖注入扩展**: AppContext 新增 `calibration_engine`, `model_evolver` 属性
- **CI 优化**: 排除 E2E 测试避免覆盖率步聚阻塞，E2E 性能阈值调整至 3s
