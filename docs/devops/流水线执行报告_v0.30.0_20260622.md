- **配置版本**: v0.30.0

### 6.2 关键配置项
```yaml
name: CI Pipeline

on:
  push:
    branches: [ main, feature/*, hotfix/* ]
  pull_request:
    branches: [ main ]

env:
  PYTHON_VERSION: '3.11'
  FORCE_JAVASCRIPT_ACTIONS_TO_NODE24: 'true'

jobs:
  code-quality:
    name: Code Quality Check
    runs-on: ubuntu-latest
    
  test:
    name: Test Suite
    runs-on: ubuntu-latest
    strategy:
      matrix:
        python-version: ['3.11', '3.12']
    
  build:
    name: Build Package
    runs-on: ubuntu-latest
    needs: [code-quality, test]
```
