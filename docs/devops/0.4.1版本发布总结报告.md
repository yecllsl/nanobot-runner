## 架构改进验证

### 定时任务配置优化

**改进内容**: 将定时任务配置从框架目录迁移到业务目录

**技术实现**:
```python
# 配置路径变更
~/.nanobot/cron/jobs.json → ~/.nanobot-runner/cron/jobs.json

# 自动迁移机制
if old_cron_store.exists() and not new_cron_store.exists():
    shutil.copy2(old_cron_store, new_cron_store)
```

**架构优势**:
- ✅ 符合AGENTS.md配置分离原则
- ✅ 业务配置存储在业务目录
- ✅ 复用nanobot框架执行器
- ✅ 提供自动迁移机制
