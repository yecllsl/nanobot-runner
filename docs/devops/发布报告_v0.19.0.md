## 2. 变更内容

### 2.1 新增功能

- **心率变异（HRV）分析**: 静息心率趋势、心率恢复分析、心率漂移检测、HRV基础统计
- **疲劳度与恢复评估**: 综合疲劳度评分、恢复状态指示器、连续训练日监控、休息日效果评估
- **身体信号智能解读**: 异常信号预警、训练建议生成、身体信号摘要
- **深度对比分析**: 周期对比、相似训练对比、训练负荷-表现关联

### 2.2 新增模块

- `src/core/analysis/hrv.py` - 心率变异分析引擎
- `src/core/analysis/fatigue.py` - 疲劳度评估引擎
- `src/core/analysis/body_signals.py` - 身体信号解读引擎
- `src/cli/commands/status.py` - 身体状态CLI命令模块

### 2.3 新增CLI命令

- `nanobotrun analysis hrv [--days 7/30/90]`
- `nanobotrun analysis hr-recovery`
- `nanobotrun analysis fatigue`
- `nanobotrun analysis recovery`
- `nanobotrun status today/weekly`
