## 发布内容

### 架构重构

将core层按功能域拆分为六大子模块，提升代码可维护性和可测试性：

| 模块 | 功能域 | 包含内容 |
|------|--------|---------|
| **base/** | 基础设施模块 | 异常体系、日志、装饰器、结果模型、Schema、上下文、用户档案 |
| **calculators/** | 计算器模块 | VDOT、比赛预测、心率、训练负荷、历史分析、伤病风险、统计聚合 |
| **config/** | 配置模块 | 配置管理、Schema、LLM配置、环境变量、备份、同步 |
| **storage/** | 存储模块 | Parquet管理、会话仓库、索引、FIT解析、导入服务 |
| **report/** | 报告模块 | 报告生成、报告服务、异常过滤 |
| **models/** | 模型模块 | 用户档案模型、训练计划模型、分析模型（从1200+行拆分） |

### 向后兼容

- 所有 `__init__.py` 提供完整公共API重导出
- 无破坏性变更，现有代码无需修改导入路径
