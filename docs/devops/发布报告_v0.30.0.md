## 2. 版本变更内容

### 核心变更
- **代码重构**: 8个超标函数拆分为职责单一子函数（gateway.py start 从37降至4个子函数）
- **类型安全**: 5处 `dict[str, Any]` 替换为 TypedDict
- **安全误报**: 11处 Bandit B105/B107 添加 nosec 注释
- **可观测性**: 3处静默异常添加 debug 日志
- **Bug修复**: 修复 nanobot-ai 0.2.1 兼容性问题（http_utils 模块不存在）
- **测试**: 4226 单元测试用例零失败，覆盖率 81%
