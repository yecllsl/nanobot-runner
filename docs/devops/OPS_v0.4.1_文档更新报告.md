**主要内容**:
- CI Pipeline失败排查
- Release Pipeline失败排查
- 权限错误排查
- 缓存问题排查
- 依赖安装问题排查
- 构建产物问题排查

**常见问题及解决方案**:

| 问题 | 症状 | 解决方案 |
|------|------|---------|
| mypy类型检查失败 | Library stubs not installed | 添加`--install-types --non-interactive`参数 |
| bandit安全扫描失败 | 检测到安全问题 | 跳过B101,B601检查或修复代码 |
| Release 403错误 | 无法创建GitHub Release | 添加`permissions: contents: write` |
| 缓存污染 | 依赖安装失败 | 更新缓存key或禁用缓存 |
