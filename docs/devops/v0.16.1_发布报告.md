## 问题与修复

### CICD失败问题

**问题**: Release workflow首次执行失败
- **失败步骤**: Check version consistency
- **根因**: README.md和CHANGELOG.md版本号仍为v0.16.0，与pyproject.toml的v0.16.1不一致
- **修复**: 更新所有文件的版本号到v0.16.1
- **验证**: 本地执行`uv run python scripts/check_version_consistency.py`通过

### 已知问题

1. **trio依赖缺失**: 38个异步测试因缺少trio模块被排除
   - 影响范围：部分agent工具测试
   - 建议：在CI环境中安装trio依赖

2. **CLI子进程测试**: 1个E2E测试因模块导入路径问题失败
   - 测试：`test_stats_command_execution`
   - 原因：子进程执行时PYTHONPATH未正确设置
   - 影响：不影响核心功能，属环境配置问题
