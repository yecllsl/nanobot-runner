## 4. v0.12.0 核心功能

### 4.1 预测规划层
- **目标达成评估引擎**: GoalPredictionEngine
  - 支持全马/半马完赛时间预测
  - 基于训练负荷和VDOT趋势的智能预测
  - 提供置信区间和风险提示
- **长期周期规划引擎**: LongTermPlanGenerator
  - 支持基础期→进展期→巅峰期→比赛期→恢复期的完整周期
  - 自动计算各阶段训练负荷和强度分布
- **智能建议引擎**: SmartAdviceEngine
  - 识别训练不足/过量风险
  - 检测有氧基础薄弱问题
  - 提供个性化调整建议

### 4.2 Agent工具扩展
- GenerateLongTermPlanTool: 长期规划工具
- EvaluateGoalAchievementTool: 目标评估工具
- GetSmartAdviceTool: 智能建议工具

### 4.3 测试覆盖

- 长期规划引擎覆盖率: 98%
