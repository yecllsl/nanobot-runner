## 4. 技术方案要点

### 4.1 TSS 计算算法

```
IF (强度因子) = (avg_hr - rest_hr) / (max_hr - rest_hr)
TSS = (duration_s * IF²) / 3600 * 100
```

### 4.2 EWMA 算法

```
ATL = EWMA(TSS, 时间窗口=7天)
CTL = EWMA(TSS, 时间窗口=42天)
TSB = CTL - ATL
```

### 4.3 性能优化策略

- 使用 Polars LazyFrame 延迟加载
- 谓词下推 (Predicate Pushdown)
- 列剪枝 (Column Pruning)
- 添加查询缓存机制
