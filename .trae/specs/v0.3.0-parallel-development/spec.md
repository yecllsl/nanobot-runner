## ADDED Requirements

### Requirement: TSS 计算功能 (T001)

系统 SHALL 提供单次跑步的 TSS (训练压力分数) 计算功能。

#### Scenario: 正常 TSS 计算

- **WHEN** 用户提供距离、时长、平均心率参数
- **THEN** 系统返回 TSS 值 (0-500)

#### Scenario: 心率数据缺失

- **WHEN** 心率数据缺失或为 0
- **THEN** 系统返回 0

#### Scenario: 时长为 0

- **WHEN** 时长参数为 0
- **THEN** 系统返回 0

#### Scenario: 计算准确性

- **WHEN** 使用标准测试数据
- **THEN** 计算结果与 TrainingPeaks 标准误差 < 5%

---

### Requirement: 配速分布分析 (T009)

系统 SHALL 提供配速分布分析功能，统计各配速区间的跑步次数和距离。

#### Scenario: 正常配速分布分析

- **WHEN** 用户请求配速分布分析
- **THEN** 系统返回各配速区间的统计信息

#### Scenario: 配速区间划分

- **WHEN** 系统进行配速分析
- **THEN** 配速区间划分合理（慢跑、轻松跑、马拉松配速、节奏跑、间歇跑）

#### Scenario: 性能要求

- **WHEN** 处理 1000 条记录
- **THEN** 计算时间 < 2 秒

---

### Requirement: 心率区间分析 (T010)

系统 SHALL 提供心率区间分析功能，基于最大心率百分比划分区间。

#### Scenario: 正常心率区间分析

- **WHEN** 用户提供心率数据和年龄参数
- **THEN** 系统返回各心率区间的统计信息

#### Scenario: 心率区间划分

- **WHEN** 系统进行心率区间分析
- **THEN** 区间划分符合运动科学原理（Z1-Z5）

#### Scenario: 自定义年龄参数

- **WHEN** 用户指定年龄
- **THEN** 最大心率按 220-年龄 计算

---

### Requirement: 训练效果评估 (T011)

系统 SHALL 提供训练效果评估功能，评估有氧和无氧效果。

#### Scenario: 正常训练效果评估

- **WHEN** 用户提供跑步数据
- **THEN** 系统返回有氧效果和无氧效果评估

#### Scenario: 恢复时间估算

- **WHEN** 系统评估训练效果
- **THEN** 提供恢复时间估算

#### Scenario: 性能要求

- **WHEN** 处理单条记录
- **THEN** 计算时间 < 2 秒

## 技术规格

### TSS 计算公式

```python
# 基于心率的 TSS 计算
IF (强度因子) = (avg_hr - rest_hr) / (max_hr - rest_hr)
TSS = (duration_s * IF²) / 3600 * 100

# 参数说明
max_hr = 220 - age  # 最大心率估算
rest_hr = 60        # 静息心率（默认值）
```

### 配速区间划分

| 区间 | 名称 | 配速范围 (秒/公里) |
|------|------|-------------------|
| Z1 | 慢跑 | > 360 (6:00+) |
| Z2 | 轻松跑 | 300-360 (5:00-6:00) |
| Z3 | 马拉松配速 | 270-300 (4:30-5:00) |
| Z4 | 节奏跑 | 240-270 (4:00-4:30) |
| Z5 | 间歇跑 | < 240 (4:00-) |

### 心率区间划分

| 区间 | 名称 | 心率范围 (% 最大心率) |
|------|------|----------------------|
| Z1 | 恢复区 | 50-60% |
| Z2 | 有氧区 | 60-70% |
| Z3 | 节奏区 | 70-80% |
| Z4 | 阈值区 | 80-90% |
| Z5 | 无氧区 | 90-100% |

### 训练效果评估标准

| 有氧效果 | 无氧效果 | 训练类型 |
|---------|---------|---------|
| 1.0-2.0 | 0-1.0 | 恢复跑 |
| 2.0-3.0 | 0-1.0 | 基础有氧 |
| 3.0-4.0 | 1.0-2.0 | 节奏跑 |
| 4.0-5.0 | 2.0-3.0 | 高强度间歇 |
