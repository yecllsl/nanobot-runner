### Task 1: T001 - 实现 TSS 计算功能

**负责模块**: `src/core/analytics.py`  
**预估工时**: 4h  
**优先级**: P0

**任务描述**:
实现单次跑步的 TSS (训练压力分数) 计算，支持基于心率的 TSS 计算。

**实现步骤**:
- [x] SubTask 1.1: 在 `AnalyticsEngine` 类中添加 `calculate_tss_for_run()` 方法
- [x] SubTask 1.2: 实现基于心率的 TSS 计算公式: `TSS = (duration_s * IF²) / 3600 * 100`
- [x] SubTask 1.3: 实现强度因子 (IF) 计算: `IF = (avg_hr - rest_hr) / (max_hr - rest_hr)`
- [x] SubTask 1.4: 添加边界条件处理（心率缺失、时长为 0）
- [x] SubTask 1.5: 编写单元测试，覆盖率 ≥ 90%

**验收标准**:
- [x] 实现 `calculate_tss_for_run()` 方法
- [x] 心率数据缺失时返回 0
- [x] 时长为 0 时返回 0
- [x] TSS 值范围合理 (0-500)
- [x] 计算结果与 TrainingPeaks 标准一致 (误差 < 5%)
- [x] 单元测试覆盖率 ≥ 90%
